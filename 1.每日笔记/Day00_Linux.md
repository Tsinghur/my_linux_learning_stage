## Vim操作

1. **移动**

   |     命令     |                 含义                 |
   | :----------: | :----------------------------------: |
   |     [n]-     |             往上移动n行              |
   |     [n]+     |             往下移动n行              |
   | :[n] 或 [n]G |             移动到第n行              |
   |      gg      |           移动光标到第一行           |
   |      G       |          移动光标到最后一行          |
   |      ^       | 移动光标到当前行的第一个非空白字符处 |
   |      0       |        移动光标到当前行的行首        |
   |      $       |              移动到行尾              |
   |      w       |      光标移动到下一个单词的词首      |
   |      b       |      光标移动到上一个单词的词首      |
   |      t)      | 向后查找，将光标移动到)的前一个字符  |
   |      f)      |       向后查找，将光标移动到)        |
   |      T"      | 向前查找，将光标移动到"的后一个字符  |
   |      F"      |       向前查找，将光标移动到"        |

2. **删除**

   | x         | 删除一个字符                                                 |
   | --------- | ------------------------------------------------------------ |
   | db        | 从光标的下一个元素删除到上一个单词的词首(上一个单词的词首会被删除) |
   | dw        | 从光标元素开始删除到下一个单词的词首(下一个单词的词首不会被删除) |
   | **diw**   | **删除光标所在的整个单词，但不包括单词周围的空格。**         |
   | **di"**   | **删除光标所在""范围内的所有内容**                           |
   | **dd**    | **删除当前这一行**                                           |
   | [n]dd     | 删除当前和下面的(n-1)行，共n行被删除                         |
   | :[m],[n]d | 删除文本的[m,n]行                                            |
   | d^        | 删除到行首(不删除光标位置的字符)                             |
   | d$        | 删除到行尾(删除光标位置的字符)                               |
   | **df)**   | **向后删除直到)，同时删除)和光标位置字符**                   |
   | **dt)**   | **向后删除直到)同时删除光标位置字符，但不删除)**             |
   | dT"       | 向前删除直到"但不删除"和光标位置字符                         |
   | dF"       | 从光标位置元素(不包含)开始向前直到"(包含)全部删掉            |

6. ==**Vim新建标签页与分屏操作**==

   1. **标签页操作**

      1. 新建标签页

         ```c
         // 新建空白标签页
         :tabnew
         // 新建空白标签页
         :tabnew main.c				
         :tabnew ~/code/test.py
         // 启动 Vim 时直接打开多个标签
         vim -p file1.c file2.h main.c
         ```

      2. 切换标签页

         ```c
         // 切换下一个标签
         gt（普通模式）
         或 :tabn
         // 切换上一个标签
         gT（普通模式）
         或 :tabp
         ```

      3. 关闭标签页

         ```c
         // 关闭当前标签
         :tabc（tab close）
         // 只保留当前标签，关闭其他所有
         :tabo（tab only）
         ```

      4. 查看

         ```c
         // 查看所有标签列表
         :tabs
         ```

   2. **分屏操作**

      1. 垂直分屏（左右）

         ```C
         :vsplit 文件名   " 或简写 :vs 文件名
         ```

         - 快捷键：`Ctrl+w` 后按 `v`
         - 效果：当前窗口左右拆分，右边打开新文件

      2. 水平分屏（上下）

         ```c
         :split 文件名    " 或简写 :sp 文件名
         ```

         - 快捷键：`Ctrl+w` 后按 `s`

      3. 在分屏间切换

         - 快捷键：`Ctrl+w` 后按 `h/j/k/l`（分别对应左 / 下 / 上 / 右）或者↑↓←→方向键
         - 或 `Ctrl+w` 后按 `w` 循环切换

      4. 调整分屏大小

         - 垂直分屏宽度：`Ctrl+w` 后按 `>`（加宽）或 `<`（变窄）
         - 水平分屏高度：`Ctrl+w` 后按 `+`（加高）或 `-`（变矮）

7. **替换**

   Vim 里替换就记一个命令：`:s`，最常用的格式直接背：

   **速记口诀**

   ```c
   当前行：:s/old/new/g
   全文：:%s/old/new/g
   确认替换：:%s/old/new/gc
   ```

   1. **替换当前行**

      ```c
      // 只替换当前行第一个匹配
      :s/旧内容/新内容/
      ```

      ```c
      // g = global，替换当前行所有匹配
      :s/旧内容/新内容/g
      ```

   2. **替换整个文件（最常用）**

      ```c
      // % = 全文
      // g = 一行里所有都替换
      :%s/旧内容/新内容/g
      ```

   3. **替换时询问确认（安全）**

      ```c
      // c = confirm，每次替换前问你：
      // y 确认 / n 跳过 / a 全部替换
      :%s/旧内容/新内容/gc
      ```

   4. **替换指定行范围**

      ```c
      // 比如替换第 5～10 行：
      :5,10s/旧内容/新内容/g
      ```

   5. 分隔符不想用 / 可以换

      ```c
      // 内容里有 / 时用 # 或 | 更方便：
      :%s#/home/user#/opt#g
      ```

## **linux的Shell命令**

1. man手册相关

   ```c
   // 看第 2 卷总说明：
   man 2 intro
   // 看第 2 卷所有函数列表：
   man -k . | grep '^.* (2)' | sort
   ```

   | 手册卷号 | 核心内容                    | 本质                  | 典型示例                           |
   | -------- | --------------------------- | --------------------- | ---------------------------------- |
   | 1        | 用户命令                    | shell命令与可执行程序 |                                    |
   | 2        | 系统调用（System Calls）    | 直接请求内核服务      | `open()`、`read()`、`fork()`       |
   | 3        | 库函数（Library Functions） | 封装的用户态函数      | `printf()`、`strcpy()`、`malloc()` |

   **部分库函数会调用系统调用（间接请求内核），但还有很多库函数完全不依赖内核，仅在用户态执行**。

2. 查看库函数源代码

   ```c
   cd coreutils-*  # 进入源码目录 // 在家目录下面
   vim src/chmod.c # 直接查看 chmod 命令源码
   ```

3. 查看内存

   ```c
   `free` 命令显示系统的内存使用情况，包括物理内存和交换分区的总量、已用量、空闲量等信息。
       
   `df` (disk free) 命令显示文件系统的磁盘空间使用情况。`df -h` 以人类可读的方式显示（GB、MB等单位）。
       
   `du` (disk usage) 命令估算文件或目录占用的磁盘空间。`du -sh *` 可以汇总显示当前目录下每个项目的大小。
   ```

4. 将VS中以**GBK**编码的main.c转换为Linux中默认的**UTF-8**编码

   ```c
   iconv -f gbk -t utf8 main.c -o main.c
   ```
