# ==什么是杨氏矩阵（杨氏矩阵不要求是方阵）==

**杨氏矩阵（Young Tableau）**

**杨氏矩阵**是一个数学概念，在组合数学和算法中都有重要应用。LeetCode 240 的矩阵正是杨氏矩阵的一种特例。

------

**1. 定义**

杨氏矩阵是一个二维矩阵，满足以下两个条件：

条件1：**行递增**

每行从左到右是**严格递增**的

条件2：**列递增**

每列从上到下是**严格递增**的

```c
[1,  4,  7,  11, 15]
[2,  5,  8,  12, 19]
[3,  6,  9,  16, 22]
[10, 13, 14, 17, 24]
[18, 21, 23, 26, 30]
```

这就是一个杨氏矩阵。

------

**2. 两种主要类型**

(1) **标准杨氏矩阵**

- 填入数字 1 到 n
- 每行每列都严格递增
- 通常用于组合数学、表示论

```c
[1, 3, 6]
[2, 5, 8]
[4, 7, 9]
```

这是 3×3 的标准杨氏矩阵。

(2) **广义杨氏矩阵**（LeetCode 用的）

- 不要求从 1 开始
- 不要求填满所有数字
- 只要求行递增、列递增

LeetCode 240 就是广义杨氏矩阵。

------

**3. 重要性质**

性质1：**从右上角搜索**

```c
当前元素 = matrix[row][col]
- 如果 target < 当前值 → 向左走（排除当前列）
- 如果 target > 当前值 → 向下走（排除当前行）
```

每次排除一行或一列，时间复杂度 **O(m+n)**。

性质2：**从左下角搜索**

类似地，往右或往上走。

性质3：**分治特性**

可以将矩阵分成四个子矩阵，利用性质进行二分查找，时间复杂度 **O(n^1.585)**。

------

**4. 杨氏矩阵 vs 普通排序矩阵**

| 特性       | 杨氏矩阵 | 普通矩阵       |
| :--------- | :------- | :------------- |
| 行递增     | ✅        | ❌              |
| 列递增     | ✅        | ❌              |
| 全局排序   | ❌        | 可能没有       |
| 搜索最优解 | O(m+n)   | 需要更复杂方法 |

**注意：** 杨氏矩阵**不是全局有序**的：

- 例如 `matrix[0][2] = 7` 和 `matrix[1][0] = 2`，虽然 `2 < 7`，但位置并不按全局顺序

------

**5.应用场景**

(1) **查找问题**（LeetCode 240）

利用角点性质快速查找

(2) **组合数学**

- 表示论的 Young 图
- 整数划分的表示

(3) **排序算法**0

- 可以用杨氏矩阵实现 **O(n log n)** 的排序
- 插入一个元素到杨氏矩阵：O(m+n)

(4) **数据存储**

- 数据仓库中的列存储优化

------

**6. 相关算法题目**

| 题目              | 说明                                          |
| :---------------- | :-------------------------------------------- |
| **LeetCode 240**  | 搜索二维矩阵 II（标准杨氏矩阵查找）           |
| **LeetCode 74**   | 搜索二维矩阵（行递增+行尾<下行头，可二分）    |
| **剑指 Offer 04** | 二维数组中的查找（同240）                     |
| **LeetCode 378**  | 有序矩阵中第 K 小的元素（可用杨氏矩阵堆解法） |

------

**7. 代码示例（杨氏矩阵查找）**

python

```
def searchYoungTableau(matrix, target):
    """
    杨氏矩阵查找
    """
    if not matrix or not matrix[0]:
        return False
    
    rows, cols = len(matrix), len(matrix[0])
    row, col = 0, cols - 1  # 从右上角开始
    
    while row < rows and col >= 0:
        if matrix[row][col] == target:
            return True
        elif target < matrix[row][col]:
            col -= 1  # 向左，排除当前列
        else:
            row += 1  # 向下，排除当前行
    
    return False
```

------

**8.总结**

**杨氏矩阵 = 行递增 + 列递增的二维矩阵**

**核心价值：**

- 利用角点性质，实现 **O(m+n)** 的查找
- 在算法面试中，看到"行递增、列递增"的矩阵，应该立刻想到杨氏矩阵和角点搜索法
- 是一个很好的**矩阵特性利用**的典型案例

LeetCode 240 就是典型的**杨氏矩阵查找问题**，最优解就是从右上角或左下角开始进行贪心搜索。

```c
// 排除法
bool searchMatrix(int** matrix, int matrixSize, int* matrixColSize, int target){
    int m = matrixSize, n = matrixColSize[0];
    int i = 0, j = n - 1; // 从右上角开始
    while(i < m && j >= 0){ // 还有剩余元素（因为只会往左下方向走，所以i有上界，j有下界）
        if(matrix[i][j] == target) // 找到target
            return true;
        else if(target > matrix[i][j])
            i++;
        else // target < matrix[i][j]
            j--;
    }

    return false;
}
```

```c
// 如果链表长度 <= size，不做任何操作，返回空结点（size即分割成2个链表且第一个链表的长度为size，若原链表长度<=size，则意味着无法分出第二个链表，所以只能返回空）

struct ListNode* splitList(struct ListNode* head, int size){
    // 先找到next_head的前一个结点
    struct ListNode* cur = head;
    for(int i = 0; i < size - 1 && cur; i++)
        cur = cur->next;

    // 如果链表长度 <= size（即cur为空或恰好cur下一个结点为空）
    if(cur == NULL || cur->next == NULL)
        return NULL; // 不做任何操作，返回空结点

    struct ListNode* next_head = cur->next;
    cur->next = NULL; // 断开next_head的前一个结点和next_head的连接
    return next_head;

    // step为步长，即参与合并的链表的长度-也即"要分割出来的第一个链表"的长度（每次分割出来两个链表，只要求第一个为长度step，剩余的组成一个链表并返回其头结点，若是分割不出来即原链表长度小于等于step则返回空）
    for(int step = 1; step < length; step *= 2){
        struct ListNode* new_list_tail = &dummy; // 新链表的末尾，初始化为哨兵节点的地址是经典的记录新链表头结点的操作
        struct ListNode* cur = dummy.next; // 每轮循环的起始结点，初始化的结果head即dummy.next
        while(cur) {
            // 从cur开始，分割出两端长为step的链表，头结点分别为head1，head2
            struct ListNode* head1 = cur;
            struct ListNode* head2 = splitList(head1, step);
            cur = splitList(head2, step); // 下一轮循环的起始结点；为什么连续分割两次：为了合并两个长为step的链表，分割两次cur正好是第2*step + 1个结点
            // 合并两段长为step的链表，合并后的头结点head，插到new_list_tail的后面（这个new_list_tail本质就是每次循环中用于储存临时新链表尾结点的指针）
            struct ListNode* tail;
            new_list_tail->next = mergeTwoLists(head1, head2, &tail);
            new_list_tail = tail; // tail在上面合并函数的调用后带回了新链表此时真正的尾结点，即拼接到新链表末尾的合并链表的尾结点
        }
    }

    return dummy.next;
}
```

```c
// 法二：最小堆
// 最小堆结构
typedef struct {
    struct ListNode** data;
    int size;
    int capacity;
} MinHeap;

// 初始化堆
MinHeap* createMinHeap(int capacity) {
    MinHeap* heap = (MinHeap*)malloc(sizeof(MinHeap));
    heap->data = (struct ListNode**)malloc(sizeof(struct ListNode*) * capacity);
    heap->size = 0;
    heap->capacity = capacity;
    return heap;
}

// 交换两个节点指针
void swap(struct ListNode** a, struct ListNode** b) {
    struct ListNode* temp = *a;
    *a = *b;
    *b = temp;
}

// 堆化（下沉）
void heapify(MinHeap* heap, int idx) {
    int smallest = idx;
    int left = 2 * idx + 1;
    int right = 2 * idx + 2;
    
    if (left < heap->size && heap->data[left]->val < heap->data[smallest]->val) {
        smallest = left;
    }
    if (right < heap->size && heap->data[right]->val < heap->data[smallest]->val) {
        smallest = right;
    }
    
    if (smallest != idx) {
        swap(&heap->data[idx], &heap->data[smallest]);
        heapify(heap, smallest);
    }
}

// 插入节点到堆中
void push(MinHeap* heap, struct ListNode* node) {
    if (heap->size >= heap->capacity) {
        // 扩容
        heap->capacity *= 2;
        heap->data = (struct ListNode**)realloc(heap->data, sizeof(struct ListNode*) * heap->capacity);
    }
    
    int i = heap->size;
    heap->data[i] = node;
    heap->size++;
    
    // 上浮
    while (i > 0 && heap->data[(i - 1) / 2]->val > heap->data[i]->val) {
        swap(&heap->data[(i - 1) / 2], &heap->data[i]);
        i = (i - 1) / 2;
    }
}

// 弹出堆顶节点
struct ListNode* pop(MinHeap* heap) {
    if (heap->size == 0) {
        return NULL;
    }
    
    struct ListNode* root = heap->data[0];
    heap->size--;
    heap->data[0] = heap->data[heap->size];
    heapify(heap, 0);
    
    return root;
}

// 检查堆是否为空
bool isEmpty(MinHeap* heap) {
    return heap->size == 0;
}

// 释放堆内存
void freeMinHeap(MinHeap* heap) {
    free(heap->data);
    free(heap);
}

struct ListNode* mergeKLists(struct ListNode** lists, int listsSize) {
    if (listsSize == 0) {
        return NULL;
    }
    
    // 创建最小堆，初始容量为 listsSize
    MinHeap* heap = createMinHeap(listsSize);
    
    // 把所有非空链表的头节点入堆
    for (int i = 0; i < listsSize; i++) {
        if (lists[i] != NULL) {
            push(heap, lists[i]);
        }
    }
    
    // 哨兵节点
    struct ListNode dummy;
    dummy.next = NULL;
    struct ListNode* cur = &dummy;
    
    // 循环直到堆为空
    while (!isEmpty(heap)) {
        struct ListNode* node = pop(heap); // 剩余节点中的最小节点
        
        if (node->next != NULL) { // 下一个节点不为空
            push(heap, node->next); // 下一个节点入堆
        }
        
        cur->next = node; // 把 node 添加到新链表的末尾
        cur = cur->next; // 准备合并下一个节点
    }
    
    freeMinHeap(heap);
    return dummy.next; // 哨兵节点的下一个节点就是新链表的头节点
}
```
